Overview of the MITE entry page

The MITE entry page can be separated in three blocks: the navigation header, the Reaction Information, and the Enzyme Information.

The naviation header allows to browse through the MITE entries. The navigation buttons can also be triggered using the 'left' and 'right' keys on a keyboard to move to the previous or the next entry, respectively. A click on the "Modify ..." button allows to propose changes.

The Enzyme Information column shows general information about the tailoring enzyme the MITE entry is about. It also shows information about Auxiliary Enyzmes (any enzymes needed for the tailoring enzyme to work properly), and a Changelog, which describes the changes this entry has undergone.

The Reaction Information column displays the known tailoring reaction of this enzyme. The Reaction SMARTS tab shows the generic reaction as expressed by the reaction SMARTS, while the Example cards below show examples of substrate-product pairs. Multiple tailoring reactions/substrat specificities can be known, and these can be selected clicking on the "Select a reaction" button.

Creating MITE-compatible reaction SMARTS and SMILES strings

Reaction SMARTS and SMILES are the core information of any MITE entry, summarizing and exemplifying the reaction and substrate specificity of the described tailoring enzyme This step-by-step guide demonstrates a convenient way to create MITE-compatible reaction SMARTS and SMILES, using freely available online tools.

What are SMILES, SMARTS, and reaction SMARTS?

SMILES and SMARTS are line notations describing chemical structures. While a SMILES string describes a molecule by specifying its atoms and their connectivity, a SMARTS string describes a search pattern of atoms and bonds. The SMILES string “C=C” describes ethylene, while the SMARTS string “[C]=[C]” matches the SMILES string of ethylene and also the SMILES string describing propylene “C=CC”. Reaction SMARTS are a powerful extension which includes both pattern-matching of substrate(s) and their modification to product(s) (“substrate>>product”). This system allows modeling chemical reactions and applying them to SMILES strings.

In MITE, reaction SMARTS are used to describe the substrate specificities and reactions of tailoring enzymes, while SMILES are used to describe the actual substrate and product of characterized reactions.

How to create reaction SMARTS and SMILES?

Writing SMILES and reaction SMARTS by hand is complicated and error-prone. Instead, we recommend to first draw the chemical structures, and then export them in their string notation.

While there are many chemistry drawing programs available, we recommend the free open-source tool Ketcher, which has an online demo version available and which will be used in this tutorial.

Why MITE?

Knowledge about reaction and substrate specificities of tailoring enzymes is often scattered in the literature. This does not only require researchers to perform time-consuming literature review, but also hampers the application of machine learning.

To close this gap, the Minimum Information about a Tailoring Enzyme (MITE) data standard and the accompanying data repository were created. The MITE data standard provides a framework to structure (meta)data in a both human- and machine-readable fashion, while the MITE data repository stores the actual data.

All data in MITE is assembled, curated, and reviewed by volunteers and can be used under a permissible open source license.

What data is stored in MITE?

MITE exclusively collects experimentally verified (meta)data on tailoring enzymes.

Each MITE entry describes a single, non-redundant enzyme-encoding gene and one or more associated tailoring reactions. These reactions are described as reaction SMARTS, which describe the substrate specificity-reaction relationship of the enzyme. These reaction SMARTS are examplified by one or more substrate(s) -> product(s) pairs in SMILES format.

Taken together, the tailoring reaction is summarized in a both human- and machine-readable fashion.

How to contribute?

MITE lives from contributions by its users (known as Contributors). Contributors can create new entries using the MITE Submission Portal, or modify existing entries in the MITE Repository. These submissions are then reviewed by topic matter experts (Reviewers) and released as updates on a monthly basis. All data on MITE is available from Zenodo for free, forever.
